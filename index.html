SdS Research Repository

A research repository for Schwarzschild-de Sitter thermodynamics, spectral/QNM analysis, and a practical interactive workbench.

This project is organized to separate exact results, empirical work, exploratory notes, and artifacts/retractions.

Simple GitHub Pages Setup

No workflow required.

  1. Run scripts/build/publish-pages-root.sh.
  2. Commit and push generated /workbench.
  3. In GitHub: Settings > Pages > Deploy from branch.
  4. Select branch main, folder /(root).

The root site is this page. The app is served from /workbench/.

This repository prioritizes transparent research status over hype. Artifact and retracted paths are retained for provenance.
