================================================================================
                        PHYSICS OVERVIEW
================================================================================

Schwarzschild-de Sitter spacetime is the unique spherically symmetric
solution to Einstein's field equations with a cosmological constant Λ > 0.
It describes a black hole embedded in an expanding universe.

Key features:
  • Two event horizons: inner (black hole) at r_b, outer (cosmological) at r_c
  • Static region between horizons where observers can exist
  • Thermodynamic entropy associated with each horizon
  • Entropy deficit Δ from the interaction between horizons
  • Finite parameter space: x ∈ (0,1) parameterizes the family

Pure de Sitter limit (x → 0):
  • Black hole shrinks away
  • Only cosmological horizon remains
  • Spacetime approaches pure de Sitter space

Nariai limit (x → 1):
  • Two horizons merge at r = 1/√Λ
  • Both temperatures approach zero
  • Entropy deficit reaches maximum (Δ/S_Λ = 1/3)
  • Both entropies equal: S_b = S_c = S_Λ/3

Important: The slider does NOT show time evolution. Each point is a distinct
spacetime. The family is a 1-parameter set of static solutions to Einstein's
field equations.

================================================================================
                      TROUBLESHOOTING
================================================================================

Q: File doesn't open in browser?
A: Try a different browser. Use Chrome, Firefox, Safari, or Edge.
   Make sure JavaScript is enabled.

Q: Validation checks showing red?
A: This shouldn't happen. Residuals should always be < 1e-10.
   Try refreshing the page.

Q: Numbers don't look right?
A: Check that you're looking at the right panel. Values are in natural
   units (c = ℏ = 1). See README.md for details.

Q: Diagrams not rendering?
A: Make sure SVG is supported (all modern browsers support it).
   Try a different browser if SVG is disabled.

Q: Performance lag?
A: This shouldn't happen. File is lightweight and optimized.
   Close other browser tabs and try again.

Q: Can't pin states?
A: Click "Pin as State A" first, then move slider, then "Pin as State B".
   Comparison table appears automatically when both are pinned.
