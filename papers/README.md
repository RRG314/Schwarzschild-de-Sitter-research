# Paper Navigation

This folder is organized as reading lanes so different readers can jump directly to what they need.

## Lanes

1. Foundational / Proved: `papers/foundational/README.md`
2. Experimental / Numerical: `papers/experimental/README.md`
3. Disproven / Retracted: `papers/disproven-or-retracted/README.md`
4. Open Questions / Next Targets: `papers/open-questions/README.md`

## Source Manuscript Locations

- Core theory manuscripts live in `papers/sds-theory/`
- Preprint-ready assets live in `papers/submitted-or-preprint-ready/`
- Draft PDFs and transition docs live in `papers/drafts/`

The lane files above provide curated entry points while preserving full provenance.
