Date: 2026-04-10

This release marks the first full presentation-grade consolidation of the SdS research program.

## What this release provides

- A complete research repository structure with clear navigation.
- A live interactive workbench on GitHub Pages.
- Explicit evidence labeling for exact, empirical, approximation-based, disproven/retracted, and open-result tracks.
- Preserved provenance and historical artifacts.
- A presentation pass focused on readability, consistency, and claim discipline.

- Negative and retracted findings remain visible and citable in context.
