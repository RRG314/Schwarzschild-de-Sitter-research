# Glossary

## SdS

Schwarzschild-de Sitter spacetime with mass parameter `M` and positive cosmological constant `Lambda`.

## Horizons `r_b`, `r_c`

- `r_b`: black-hole horizon radius.
- `r_c`: cosmological horizon radius.

## Entropies `S_b`, `S_c`

Entropy values associated with `r_b` and `r_c`.

## Deficit `Delta`

A derived quantity used in this project to track entropy-structure relations.

## Temperatures `T_b`, `T_c`

Horizon temperatures for black-hole and cosmological horizons.

## Carnot efficiency `eta_C`

Efficiency-style quantity derived from horizon temperature ratio in the project framework.

## QNM

Quasinormal modes. In this repository, spectral/QNM claims are separated into exact, approximation-based, empirical, and artifact categories.

## Nariai limit

Parameter boundary where SdS horizons approach coincidence. Used as a safety boundary in evolution experiments.

## Evolution mode (app)

A simplified semiclassical parameter evolution tool using `dM/dt = ±k/M^2` and exact SdS reconstruction per step.
